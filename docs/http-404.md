# HTTP 404

> 404 Not Found

![](https://http.cat/404)

HTTP 404 상태 코드는 클라이언트가 요청한 리소스를 서버에서 찾을 수 없거나, 존재한다는 사실을 공개하고 싶지 않음을 나타낸다.
