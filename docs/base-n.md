# 진법

## 2진법

- 예시: 1011 0110
- to 8:
  ```
  10    110     110
  2+0 | 4+2+0 | 4+2+0
  = 2 | = 6   | = 6
  1011 0110 = 266
  ```
- to 10:
  ```
  1011        0110
  128+0+32+16+0+4+2+0
  = 182
  ```
- to 16:
  ```
  1011      0110
  8+0+2+1 | 0+4+2+0
  = 11    | = 6
  = B     |
  1011 0110 = B6
  ```
