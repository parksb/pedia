# Your Brain on ChatGPT: Accumulation of Cognitive Debt when Using an AI Assistant for Essay Writing Task

::: INFO
[Nataliya Kosmyna et al., 『Your Brain on ChatGPT: Accumulation of Cognitive Debt when Using an AI Assistant for Essay Writing Task』, 2025.](https://arxiv.org/abs/2506.08872)
:::

LLM을 활용한 글쓰기가 인간의 인지 능력에 미치는 영향에 관한 연구. 핵심 주장은 [[artificial-intelligence]]{AI}에 의존하는 글쓰기가 인지 부채를 축적시킨다는 것이다.

피실험자를 세 그룹으로 나눠 동일한 조건의 글쓰기 세션을 3회 반복했다: ChatGPT를 활용해 에세이를 작성한 LLM 그룹, 검색엔진만 활용해 작성한 Search Engine 그룹, 도구없이 순수 사고로 작성한 Brain-only 그룹. 네 번째 세션에서는 LLM 그룹을 Brain-only 조건으로 전환(LLM-to-Brain 그룹)했고, Brain-only 그룹은 LLM 조건으로 전환(Brain-to-LLM 그룹)했다.

실험 결과는 다음과 같다.

- 뇌 연결망의 강도는 도구 사용에 반비례했다. LLM 그룹은 인지 활동이 현저히 감소한 모습을 보였다. 네 번째 세션에서 교차 실험한 결과 LLM-to-Brain 그룹은 연결성이 감소했다. 뇌가 갑자기 혼자 작업해야 할 때 저조한 참여 상태를 보였다. AI 의존 후 독립적 사고 능력이 위축된 것으로 보인다. Brain-to-LLM 그룹은 기억 회상과 후두-두정엽, 전전두엽 활성화가 높게 유지됐다. 뇌의 독립적 사고 훈련이 긍적적 이월 효과를 보였다.
- LLM 그룹에서 작성한 에세이를 NLP 분석한 결과 동질성을 보였다. 즉, AI가 생성한 에세이는 개인차가 줄고 획일화되는 경향이 있었다.
- 에세이에 대한 소유감(ownership)은 LLM 그룹이 가장 낮았고, Brain-only 그룹이 가장 높았다. LLM 그룹은 자신이 제출한 에세이를 정확히 인용하는 데 어려움을 겪었다.

LLM 사용자들은 4개월에 걸친 실험에서 일관되게 신경학적, 언어적, 행동적 수준 모두에서 저조한 성과를 보였다. AI가 즉각적인 편의성을 제공하지만, 장기적으로 인지 능력의 발달을 저해할 수 있으며, 특히 교육 환경에서 LLM에 대한 과도한 의존은 학습 자체의 신경학적 기반을 약화시킬 수 있다. 연구는 이를 인지 부채의 누적으로 해석한다.
