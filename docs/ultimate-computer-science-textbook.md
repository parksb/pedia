# 궁극의 컴퓨터과학 교과서

컴퓨터 과학의 모든 것을 다루는 교과서.

## 범위

교과서가 컴퓨터과학의 범위를 어떻게 설정할 것인지가 중요하다. 너무 저수준까지 다루면 반도체 공정부터 설명해야 하고, 너무 고수준까지 다루면 바이브 코딩하는 방법까지 설명해야 한다.
